OPCIONAL (solo si lo quieren usar o cambiar)
================================================================================

  Variable: VITE_LOCATION_LAT y VITE_LOCATION_LNG
            → Ya apuntan a Ibarbálz 1331 (~ -31.407736, -64.173544). Solo cambiar si mueven el pin.

  Variable: VITE_LOCATION_DIRECTIONS
            → Texto más detallado de cómo llegar, estacionamiento, portón, etc.

  Variable: VITE_EVENT_ADDRESS / VITE_LOCATION_ADDRESS
            → Si quieren agregar calle, número y ciudad además del nombre del complejo.

  Variable: VITE_BANK_CBU / VITE_BANK_NAME
            → Solo si quieren mostrar CBU o banco además del alias.

  Variable: VITE_GROOM_NAME          → Suele estar vacío en baby shower (OK).

  Variable: VITE_RSVP_EMAIL_ENABLED y VITE_RSVP_EMAIL_ADDRESS
            → Solo si habilitan confirmación por correo además de WhatsApp.

  Variable: VITE_FAMILY_MESSAGE      → Mensaje familiar personalizado (varios párrafos);
            si no lo mandan, la página usa los textos por defecto.

  Variable: VITE_SOCIAL_HASHTAG (solo hashtag; sin Instagram)
            → Si no tienen usuario todavía, pueden dejar el de ejemplo o un guion hasta definir.
